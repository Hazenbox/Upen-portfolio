# Groq Free API Key Setup

## How to Get Your Free Groq API Key

1. **Visit Groq Console**: Go to https://console.groq.com/
2. **Sign Up**: Create a free account (no credit card required)
3. **Get API Key**: 
   - Go to API Keys section
   - Click "Create API Key"
   - Copy your API key

## Add to Your Project

1. **Create `.env` file** in the root directory (if it doesn't exist)
2. **Add your API key**:
   ```
   GROQ_API_KEY=your_api_key_here
   ```

## Free Tier Limits

Groq offers a **generous free tier**:
- Fast inference (very low latency)
- Multiple model options
- No credit card required
- Perfect for portfolio projects

## Alternative Free Options

If you prefer other providers:

1. **Hugging Face** (https://huggingface.co/)
   - Free inference API
   - Many open-source models

2. **Together.ai** (https://together.ai/)
   - Free tier available
   - Good model selection

3. **Vercel AI Gateway** (if deployed on Vercel)
   - Unified API for multiple providers
   - Free tier available
